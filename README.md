# Claude Usage Bars 🚀

Chrome Extension สำหรับมอนิเตอร์โควตาการใช้งาน Claude.ai (ทั้งแบบ 5 ชั่วโมง และรายสัปดาห์) บนแถบเครื่องมือเบราว์เซอร์ของคุณ โดยคำนวณเวลานับถอยหลัง (Countdown) ให้แบบเรียลไทม์!

## ✨ ฟีเจอร์หลัก
- **Zero-Configuration:** ไม่ต้องใช้ API Key หรือ Token แค่ติดตั้งแล้วล็อกอินหน้าเว็บ Claude.ai ก็ใช้งานได้ทันที (ดึงข้อมูลผ่านระบบ Session Cookies อย่างปลอดภัย)
- **Real-time Countdown:** แสดงเวลานับถอยหลังก่อนที่โควตาจะรีเซ็ตวินาทิต่อวินาที
- **Smart Notifications:** แจ้งเตือนผ่านระบบ Chrome Notification เมื่อโควตาใช้งานใกล้เต็ม (>= 95%)
- **Cyberpunk Dark Mode:** หน้าตา UI สวยงาม ออกแบบสไตล์ธีมมืด JetBrains Mono อ่านง่าย สบายตา
- **Manual Refresh:** มีปุ่มกดสั่งดึงข้อมูลล่าสุดด้วยมือได้ทันที

## 🛠️ วิธีการติดตั้ง (Developer Mode)
1. กดดาวน์โหลดซอร์สโค้ดนี้ลงเครื่องคอมพิวเตอร์ของคุณแล้วแตกไฟล์ (Unzip)
2. เปิด Google Chrome แล้วเข้าไปที่ URL: `chrome://extensions/`
3. เปิดโหมดนักพัฒนาที่มุมขวาบน (**Developer mode**)
4. คลิกปุ่ม **Load unpacked** (โหลดส่วนขยายที่คลายแพ็กแล้ว) ที่มุมซ้ายบน
5. เลือกโฟลเดอร์โปรเจกต์นี้
6. เข้าใช้งานหน้าเว็บ [Claude.ai](https://claude.ai) และรีเฟรชหน้าเว็บ 1 ครั้งเพื่อให้ส่วนขยายเริ่มบันทึกค่าและทำงาน!

- **Beware of Malicious Forks:** เนื่องจากโปรเจกต์นี้เปิดเผยซอร์สโค้ด (Open Source) ผู้อื่นสามารถคัดลอก (Fork) ไปดัดแปลงได้ โปรดตรวจสอบโค้ดในไฟล์ `background.js` และ `popup.js` เสมอหากคุณนำโค้ดมาจากแหล่งอื่น เพื่อให้มั่นใจว่าไม่มีการเพิ่มคำสั่งดักจับคุกกี้ (Cookies) หรือส่งข้อมูลส่วนตัวของคุณไปยังเซิร์ฟเวอร์ภายนอก (Third-party)
